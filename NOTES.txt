Download data
=============

Preprocess data
===============
* Download URL for PLINK: http://www.cog-genomics.org/plink2
* Commands to set up PLINK 1.9b5.2:
    cd bin
    wget http://bit.ly/2nUxOQX -O plink.zip // Download 64-bit Linux binary.
    unzip plink.zip                         // Uncompress ZIP file.

Admixture analysis
==================
* Download URL: https://www.genetics.ucla.edu/software/admixture
* Commands to set up ADMIXTURE 1.3.0 on midway2:
    cd bin
    wget http://bit.ly/2BNrPa8 -O admixture.tar.gz // Download Linux binary.
    tar zxvf admixture.tar.gz                      // Unpack archive.
    mv admixture_linux-1.3.0/admixture .
